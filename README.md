## 云档3.0版本

> 由于之前的2.0版本太多坑，代码已被我无情的抛弃，现着手弄3.0版本。

基于taro开发


安装 Taro 开发工具 @tarojs/cli

使用 npm 或者 yarn 全局安装，或者直接使用npx
```shen
yarn global add @tarojs/cli
git clone https://github.com/SmallRuralDog/yundocs.git xxx
cd xxx
yanr install
yarn run dev:weapp
```
`xxx`是你的项目名称，可随意更改
